## 戳一戳
现在戳一戳事件支持监听了。
相关事件类型：
- `MiraiGroupNudgeEvent` -> 群成员（群内）戳一戳
- `MiraiMemberNudgeEvent` -> 群成员（临时会话）戳一戳
- `MiraiFriendNudgeEvent` -> 好友戳一戳
- `MiraiStrangerNudgeEvent` -> 陌生人戳一戳
